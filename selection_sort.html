選擇排序法 (Selection Sort)

選擇排序是一種簡單的排序算法。

算法原理

選擇排序反覆地從未排序部分選擇最小元素，放到已排序部分的末尾。

JavaScript 實現

function selectionSort(arr) { let n = arr.length; for (let i = 0; i < n-1; i++) { let minIdx = i; for (let j = i+1; j < n; j++) { if (arr[j] < arr[minIdx]) { minIdx = j; } } if (minIdx !== i) { [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]]; } } return arr; } // 範例使用 let array = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", array); selectionSort(array); console.log("排序後:", array);

Python 實現

def selection_sort(arr): n = len(arr) for i in range(n-1): min_idx = i for j in range(i+1, n): if arr[j] < arr[min_idx]: min_idx = j if min_idx != i: arr[i], arr[min_idx] = arr[min_idx], arr[i] return arr # 範例使用 array = [64, 34, 25, 12, 22, 11, 90] print("原始陣列:", array) selection_sort(array) print("排序後:", array)
