## 氣泡排序法
氣泡排序是一種簡單的排序算法，它反覆地比較相鄰元素，如果順序錯誤就交換它們。

## 選擇排序法
選擇排序反覆地從未排序部分選擇最小元素，放到已排序部分的末尾。
就像從一堆數字中不斷選出最小的放到前面一樣。

## 插入排序法
插入排序將元素插入到已排序部分的正確位置。
就像打撲克牌時不斷將新牌插入到已排序的手牌中。

## 合併排序法
合併排序使用分治法，將陣列分成兩半，遞歸排序後合併。
就像將一堆紙牌分成兩堆，各自排序後再合併成有序的一堆。

## 綜合比較
氣泡、選擇、插入排序：
都屬於簡單排序，時間複雜度為 O(n²)，適合資料量小；其中插入排序在「幾乎已排序」時最快。

合併排序：
採分治法，時間複雜度為 O(n log n)，效率較高，適合大量資料，但需要額外空間。
