插入排序法 (Insertion Sort)

插入排序是一種簡單的排序算法。

算法原理

插入排序將元素插入到已排序部分的正確位置。

JavaScript 實現

function insertionSort(arr) { let n = arr.length; for (let i = 1; i < n; i++) { let key = arr[i]; let j = i - 1; while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j--; } arr[j + 1] = key; } return arr; } // 範例使用 let array = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", array); insertionSort(array); console.log("排序後:", array);

Python 實現

def insertion_sort(arr): n = len(arr) for i in range(1, n): key = arr[i] j = i - 1 while j >= 0 and arr[j] > key: arr[j + 1] = arr[j] j -= 1 arr[j + 1] = key return arr # 範例使用 array = [64, 34, 25, 12, 22, 11, 90] print("原始陣列:", array) insertion_sort(array) print("排序後:", array)
