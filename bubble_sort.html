氣泡排序法 (Bubble Sort)

氣泡排序是一種簡單的排序算法。

算法原理

氣泡排序反覆地比較相鄰元素，如果順序錯誤就交換它們。

JavaScript 實現

function bubbleSort(arr) { let n = arr.length; for (let i = 0; i < n-1; i++) { for (let j = 0; j < n-i-1; j++) { if (arr[j] > arr[j+1]) { // 交換 let temp = arr[j]; arr[j] = arr[j+1]; arr[j+1] = temp; } } } return arr; } // 範例使用 let array = [64, 34, 25, 12, 22, 11, 90]; console.log("原始陣列:", array); bubbleSort(array); console.log("排序後:", array);

Python 實現

def bubble_sort(arr): n = len(arr) for i in range(n-1): for j in range(0, n-i-1): if arr[j] > arr[j+1]: arr[j], arr[j+1] = arr[j+1], arr[j] return arr # 範例使用 array = [64, 34, 25, 12, 22, 11, 90] print("原始陣列:", array) bubble_sort(array) print("排序後:", array)
